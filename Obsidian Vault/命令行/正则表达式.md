| 字符          | 内容                                     |
| ------------- | ---------------------------------------- |
| `\d`          | 单个数字                                 |
| `.`           | 任意单个字符                             |
| `\.`          | 字符.                                    |
| `[abc]`       | 单个字符,可以是a或b或c                   |
| `[^abc]`      | 单个字符,不可以是a或b或c                 |
| `[a-z],[0-6]` | 单个字符,用范围设置                      |
| `\w`          | 单个英文or数字                           |
| `\W`          | 单个非英文or数字                         |
| `a{1,3}`      | 匹配a至少1次,至多3次,a可以替换成其他转义 |
| `a*`          | 有0至多个a字符, a可以替换成其他转义      |
| `a+`          | 有1至多个a字符, a可以替换成其他转义      |
| `a?`          | 在匹配时可以略过a, 可以替换成其他转义    |
| `\s`          | 任意长度的空格                           |
| `^abc`        | 以a开头                                  |
| `abc$`        | 以c结尾                                  |
| `(a)`         | 包部分字符,仅提取该部分                  |
| `(a(b))`      | 嵌套提取                                 |
| `(a\|b\|c)`   | 多个条件选择,分隔符为\|                  |

用于`awk` 等文本操作软件
