## 什么是文献笔记？
- ❌不是批注， 高亮和勾画 => 一段时间后会遗忘批注目的和内容
- ❌不是调研报告或展示 => 文献笔记是给自己看的
- ❌不是文献综述 => 不需要像综述一样详细
- ✔文献元数据（标题， 作者等） => 方便引用回看
- ✔背景、 贡献、 方法、 结果 => 归纳文献内容
- ✔自己的想法与归纳总结 => 记录想法

**文献元数据（笔记模板，item插入）+文献笔记（双链链接到文献笔记）**

## 文献导入
1. 浏览器插件（右上角）
2. Zotero内，绿色加号右边通过链接（arxiv，DOI）等添加
3. 拖入PDF
4. 数据库选择Refworks格式，点击“复制到剪贴板”，Zotero通过快捷键Ctrl+Shift+Alt+I（从剪贴板导入）
5. 批量导入：数据库内批量获得Refworks并导入，可按住Shift+鼠标左键，将文献题录移动到其它目录下。

## 管理题录
1. 复制分类：鼠标拖动
2. 移动分类：Shift+鼠标拖动
3. 查重：重复文献
4. 关联：添加关联文献
5. 标签：添加标签，同时可选颜色
6. RSS订阅：自动抓取期刊文章
7. +（加号） 展开所有目录，-（减号）折叠所有目录
8. 数字1或2或3等，快速设置标签
